## Замечания

1. **NumPy/Matplotlib**: Для построения графиков требуется установка:
   ```bash
   pip3 install --user numpy matplotlib
   ```
   или использование виртуального окружения.

2. **Количество процессов**: На локальной машине может быть ограничено количество доступных слотов для процессов. Для больших экспериментов рекомендуется использовать кластер или увеличить количество слотов через параметры mpirun.

3. **Размеры данных**: Все программы корректно обрабатывают различные размеры входных данных и количества процессов.
